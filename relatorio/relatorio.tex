\begin{abstract}
O problema de N-corpos gravitacional é um sistema de equações diferenciais ordinárias numericamente custoso de se resolver, com custo quadrático. Neste trabalho testarei a implementação do método Parareal para alguns problemas de valor inicial distintos: órbitas periódicas, órbitas não periódicas e suaves, e órbitas não periódicas com choques elásticos. Para isso, métodos de Runge-Kutta e integradores simpléticos são implementados, visando comparar o comportamento das integrais primeiras do sistema sob o método paralelo.
\end{abstract}
